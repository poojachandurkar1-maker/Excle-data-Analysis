# Swiggy Sales & Restaurant Analysis Dashboard

## Project Overview

This project is an Excel-based data analysis dashboard created using Swiggy restaurant and food order data. The project focuses on analyzing restaurant performance, food categories, ratings, pricing trends, and customer preferences across different cities and states.

The dashboard helps users understand:

- Restaurant performance
- Popular food categories
- Pricing trends
- Customer ratings
- City-wise and state-wise analysis
- Order trends by date and week

## Problem Statement

The goal of this project is to analyze real-world Swiggy sales data and create an interactive Excel dashboard that helps understand sales performance, customer satisfaction, food trends, and regional business insights.

The dashboard is designed to support business decision-making by tracking important KPIs and identifying patterns in customer orders and restaurant performance.

---

## 1. Total Sales

Overall revenue generated from food orders.

## 2. Average Rating

Measures customer satisfaction levels across all restaurants.

## 3. Average Order Value

Represents average revenue generated per order.

## 4. Rating Count

Total number of customer reviews received.

## 5. Total Orders

Total number of food orders received.
